NBA Statistics Database

This project includes a full data collection and implementation of the National Basketball Association (NBA), it’s players and the statistics from the last twenty years. This information is fundamental to the league and to its fans. These statistics can be used to compare player productivity from a spread of years, compare players, rank teams, and effectively find the value of the player to a franchise. The motivation behind this project my interests in basketball and having been a fan for many years this project was the perfect fit. We wanted to make it easy to find statistics of any player and compare and contrast teams for user’s needs.

Arduino VR Headset

With the use of Arduino and Unity Engine, I built my own virtual reality system as an introduction into the field. I used a availible plugin to unity call unduino to communcate the analog inputs from a 9 degrees of freedom breakout sensor and joystick. This information was passed from the arduino microprocessor to the Unity Engine through use of serial ports. I learned from this project that this was a slow communcation and often lagged but was a very exciting project to work on and research.

Sleep Tracker Project

This website was created to take in data about a users sleep and create visuals for the user. I implemented a login system that tracked if the user had an account and verify the password entered. the user can also register for an account and has error handling to make sure the account is set up correctly. Registering for an account also hashes your passwords so it is not in plain text in the database. This project taught me a lot about PHP and security of website communicating with a database. this is one of my most enjoyable projects that gave me a lot of satisfaction upon completion. I do hope to implement graphs and account pages to further my learning.

Guidence System

My group of three others and myself were tasked with designing and creating a guidence system that found that displayed and interface to show the user where to go next in a given city. The applicatoin allows for a user to enter in as many addresses as they need in their route, much like a paper boy would need. The application runs and finds the shortest next route, outputs a line on the map and a dialogue of directions.
